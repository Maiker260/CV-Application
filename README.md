
# React CV Builder Project

A dynamic **CV builder application** built with React, designed to help users create, edit, and display their curriculum vitae.



## Installation

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone git@github.com:Maiker260/CV-Application.git
   ```

2. **Install dependencies**
   ```bash
   cd CV-Application
   npm install
   ```

3. **Build and run**
   ```bash
   npm run dev
   ```

4. **Open the app**  
    
## Features

- **General Information Section**
  - Input fields for **Name**, **Email**, and **Phone Number**.
  - Edit and Submit buttons.
  - Submitted data displayed as HTML elements.

- **Educational Experience Section**
  - Input fields for **School/Institution**, **Degree/Career/Course**, and **Date**.
  - Edit and Save functionality to display and update data dynamically.

- **Practical Experience Section**
  - Input fields for **Company Name**, **Position Title**, **Description**, **Start Date**, and **End Date**.
  - Edit and Save buttons for modifying and displaying job information.

- **Dynamic Editing**
  - Users can switch between **editing** and **displaying** content.
  - All input values persist during editing, allowing updates and resubmission.

- **State Management**
  - Heavy use of **React state** and **props** for dynamic updates.
  - Modular component-based structure for maintainability.
